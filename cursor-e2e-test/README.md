# cursor-e2e-test

An end-to-end learning playground for practising the full workflow: project setup, Git, database, GitHub, Vercel, and collaboration — all inside Cursor.

Features will be added step by step. This repo is the clean foundation.

---

## Setup

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser. You should see a page with the project name.

---

## Stack

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Deploys to [Vercel](https://vercel.com) with zero configuration
